Projet visant à l'utilisationt des connaissances obtenue durant la formation Handigital.

Le sujet est une application format desktop visant a aider les personnes à trouver des camarade pour les jeux multijoueur.

Il existe un role admin permettant d'apporter des modification sans recourir à la BDD directement (en travaux)

Les recommendation de jeux, amis et salons ainsi que les possibles classements se feront grace a l'aléatoire grace à la fonction PHP shuffle().




Note:
la base de donnees est "projethandigital.sql"
de plus un utilisateur "projet_user" avec le mots de passe: "projet_password" sont necessaire pour acceder à la BDD sans toucher au code.
la modification des information concernant la BDD se fait dans "bdd/connect.php".
